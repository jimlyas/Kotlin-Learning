2.4 Stack

Learn about LIFO data structure with O(1) push/pop operations and practical applications.

What is a Stack?

A stack is a LIFO (Last In, First Out) data structure. Think of it like a stack of plates - you can only add or remove from the top.

Operations:

• push - Add element to top (O(1))

• pop - Remove element from top (O(1))

• peek - View top element without removing (O(1))

Real-World Use Cases

Browser Back Button

Android and web browsers use stacks to track visited pages for back navigation.

Undo/Redo

Text editors use stacks to track changes for undo functionality.

Expression Evaluation

Compilers use stacks to evaluate mathematical expressions and parentheses matching.

JVM Stack Frames

Java/Kotlin method calls use call stacks to manage function execution.

Stack Implementation

Kotlin
fun isValid(s: String): Boolean {
    val stack = ArrayDeque<Char>()
    val map = mapOf('(' to ')', '[' to ']', '{' to '}')
    
    for (char in s) {
        if (map.containsKey(char)) {
            stack.addLast(char)
        } else if (map.containsValue(char)) {
            if (stack.isEmpty() || map[stack.last()] != char) {
                return false
            }
            stack.removeLast()
        }
    }
    return stack.isEmpty()
}

ArrayDeque Functional Operations

Kotlin's ArrayDeque implements Deque and supports operations on both ends. When used as a stack, use addLast() for push and removeLast() for pop.

Kotlin - ArrayDeque operations
fun main() {
    val stack = ArrayDeque<Int>()
    
    // Stack operations (LIFO)
    stack.addLast(1)   // push - O(1)
    stack.addLast(2)
    stack.addLast(3)
    println(stack.last())   // peek - 3
    println(stack.removeLast())  // pop - 3, O(1)
    println(stack.removeLast())  // pop - 2, O(1)
    
    // Queue operations (FIFO) - can also use as queue
    val queue = ArrayDeque<String>()
    queue.addLast("a")
    queue.addLast("b")
    queue.addLast("c")
    println(queue.removeFirst())  // "a" - O(1)
    println(queue.removeFirst())  // "b" - O(1)
    
    // Conversion to List
    val deque = ArrayDeque(listOf(1, 2, 3))
    val asList = deque.toList()
    val reversedList = deque.toList().reversed()
    
    // Iterate (note: iteration order varies)
    for (item in deque) {
        print("$item ")
    }
}

LeetCode: Valid Parentheses

Easy

Given a string s containing just the characters '(', ')', '{', '}', '[' and ']', determine if the input string is valid. An input string is valid if: Open brackets must be closed by the same type of brackets, and open brackets must be closed in the correct order.

View on LeetCode

Use a stack. Push opening brackets onto the stack. When you encounter a closing bracket, check if it matches the top of the stack.

Kotlin Solution
fun isValid(s: String): Boolean {
    val stack = ArrayDeque<Char>()
    val matching = mapOf('(' to ')', '[' to ']', '{' to '}')
    
    for (c in s) {
        if (c in matching) stack.addLast(c)
        else {
            if (stack.isEmpty() || matching[stack.last()] != c) return false
            stack.removeLast()
        }
    }
    return stack.isEmpty()
}
