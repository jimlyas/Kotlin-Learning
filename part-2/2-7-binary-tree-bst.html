2.7 Binary Tree / BST

Learn about hierarchical data structures with O(log n) search in balanced trees.

Binary Search Tree

A BST is a binary tree where left children are smaller than the parent and right children are larger. This ordering enables O(log n) search in balanced trees.

Operations:

• Search: O(log n) average, O(n) worst (skewed tree)

• Insert: O(log n) average

• Delete: O(log n) average

Real-World Use Cases

File Systems

Directory structures in OS use tree hierarchies for organization.

XML/DOM Parsing

HTML and XML documents are parsed into tree structures.

Database Indexes

B-trees (balanced variant) power database indexing for fast queries.

Expression Parsers

Compilers use Abstract Syntax Trees for parsing code.

BST Implementation

Kotlin
class TreeNode(var `val`: Int) {
    var left: TreeNode? = null
    var right: TreeNode? = null
}

fun maxDepth(root: TreeNode?): Int {
    if (root == null) return 0
    val leftDepth = maxDepth(root.left)
    val rightDepth = maxDepth(root.right)
    return maxOf(leftDepth, rightDepth) + 1
}

Tree Traversals with Functional Operations

Binary trees can be traversed and processed using functional operations. Kotlin's sequence operations enable elegant tree processing.

Kotlin - Tree traversal operations
data class TreeNode(
    val value: Int,
    val left: TreeNode? = null,
    val right: TreeNode? = null
)

fun main() {
    val root = TreeNode(1, 
        TreeNode(2, TreeNode(4), TreeNode(5)),
        TreeNode(3, TreeNode(6), TreeNode(7))
    )
    
    // Collect all values using DFS
    fun collect(node: TreeNode?): List<Int> {
        if (node == null) return emptyList
        return listOf(node.value) + collect(node.left) + collect(node.right)
    }
    
    val allValues = collect(root)
    println(allValues)  // [1, 2, 4, 5, 3, 6, 7]
    
    // Filter values greater than 3
    val filtered = allValues.filter { it > 3 }
    println(filtered)  // [4, 5, 6, 7]
    
    // Map to doubled values
    val doubled = allValues.map { it * 2 }
    println(doubled)  // [2, 4, 8, 10, 6, 12, 14]
    
    // Sum all values
    val sum = allValues.reduce { acc, n -> acc + n }
    println(sum)  // 28
}
Kotlin - Level order with map
fun main() {
    val root = TreeNode(1, 
        TreeNode(2, TreeNode(4), TreeNode(5)),
        TreeNode(3, TreeNode(6), TreeNode(7))
    )
    
    // BFS using Queue - collect by level
    fun levelOrder(root: TreeNode?): ListInt>> {
        if (root == null) return emptyList
        
        val result = mutableListOfInt>()
        val queue = ArrayDeque<TreeNode>()
        queue.addLast(root)
        
        while (queue.isNotEmpty()) {
            val level = mutableListOf<Int>()
            val size = queue.size
            
            for (i in 0 until size) {
                val node = queue.removeFirst()
                level.add(node.value)
                node.left?.let { queue.addLast(it) }
                node.right?.let { queue.addLast(it) }
            }
            result.add(level)
        }
        return result
    }
    
    val levels = levelOrder(root)
    println(levels)  // [[1], [2, 3], [4, 5, 6, 7]]
    
    // Process each level with map/filter
    val maxInEachLevel = levels.map { it.maxOrNull() }
    println(maxInEachLevel)  // [1, 3, 7]
    
    val levelWithEven = levels.indexOfFirst { it.any { n -> n % 2 == 0 } }
    println(levelWithEven)  // 1
}

LeetCode: Maximum Depth of Binary Tree

Easy

Given the root of a binary tree, return its maximum depth. A binary tree's maximum depth is the number of nodes along the longest path from the root node down to the farthest leaf node.

View on LeetCode

Use recursion. The depth of the current node is 1 + max(depth of left subtree, depth of right subtree). Base case: null node has depth 0.

Kotlin Solution
fun maxDepth(root: TreeNode?): Int {
    if (root == null) return 0
    return maxOf(maxDepth(root.left), maxDepth(root.right)) + 1
}
