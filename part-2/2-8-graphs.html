2.8 Graphs

Learn about graph representations, traversal algorithms (BFS/DFS), and real-world applications.

What is a Graph?

A graph consists of vertices (nodes) connected by edges. Graphs can be directed or undirected, weighted or unweighted.

Representations:

• Adjacency Matrix: O(1) edge lookup, O(V²) space

• Adjacency List: O(V + E) space, efficient for sparse graphs

Real-World Use Cases

Social Networks

Facebook, LinkedIn use graphs to represent friendships and connections.

GPS Navigation

Google Maps uses weighted graphs to find shortest routes.

Web Page Ranking

Google's PageRank algorithm operates on web link graphs.

Network Routing

Internet routing protocols use graph algorithms.

LeetCode: Number of Islands

Medium

Given an m x n 2D binary grid which represents a map of '1's (land) and '0's (water), return the number of islands. An island is surrounded by water and is formed by connecting adjacent lands horizontally or vertically.

View on LeetCode

Use DFS or BFS to explore connected land cells. When you find a '1', mark all connected '1's as visited and increment the island count.

Kotlin Solution (DFS)
fun numIslands(grid: Array<CharArray>): Int {
    var count = 0
    for (i in grid.indices) {
        for (j in grid[0].indices) {
            if (grid[i][j] == '1') {
                count++
                dfs(grid, i, j)
            }
        }
    }
    return count
}

fun dfs(grid: Array<CharArray>, i: Int, j: Int) {
    if (i < 0 || i >= grid.size || j < 0 || j >= grid[0].size || grid[i][j] == '0') return
    grid[i][j] = '0'
    dfs(grid, i + 1, j); dfs(grid, i - 1, j)
    dfs(grid, i, j + 1); dfs(grid, i, j - 1)
}

Graph Operations with Collections

Graphs in Kotlin are often represented using Maps and Lists, enabling powerful functional operations for processing vertices and edges.

Kotlin - Adjacency list representation
fun main() {
    // Adjacency list using Map
    val graph = mapOf(
        "A" to listOf("B", "C"),
        "B" to listOf("A", "D"),
        "C" to listOf("A", "D"),
        "D" to listOf("B", "C")
    )
    
    // Get neighbors of a vertex
    val neighborsOfA = graph["A"] ?: emptyList()
    println(neighborsOfA)  // [B, C]
    
    // Get all vertices
    val vertices = graph.keys.toList()
    println(vertices)  // [A, B, C, D]
    
    // Get all edges (as flat list)
    val allEdges = graph.values.flatMap { it }
    println(allEdges)  // [B, C, A, D, A, D, B, C]
    
    // Unique vertices from edges
    val uniqueNeighbors = allEdges.toSet()
    println(uniqueNeighbors)  // {A, B, C, D}
}
Kotlin - Graph algorithms with functional ops
fun main() {
    val graph = mapOf(
        0 to listOf(1, 2),
        1 to listOf(0, 2),
        2 to listOf(0, 1, 3),
        3 to listOf(2)
    )
    
    // BFS using queue
    fun bfs(start: Int): List<Int> {
        val visited = mutableSetOf<Int>()
        val result = mutableListOf<Int>()
        val queue = ArrayDeque<Int>()
        
        queue.addLast(start)
        visited.add(start)
        
        while (queue.isNotEmpty()) {
            val vertex = queue.removeFirst()
            result.add(vertex)
            
            val neighbors = graph[vertex] ?: emptyList()
            for (neighbor in neighbors) {
                if (neighbor !in visited) {
                    visited.add(neighbor)
                    queue.addLast(neighbor)
                }
            }
        }
        return result
    }
    
    val bfsResult = bfs(0)
    println(bfsResult)  // [0, 1, 2, 3]
    
    // Find path using filter and map
    fun findConnected(start: Int): Set<Int> {
        val visited = mutableSetOf<Int>()
        val stack = ArrayDeque<Int>()
        stack.addLast(start)
        
        while (stack.isNotEmpty()) {
            val v = stack.removeLast()
            if (v !in visited) {
                visited.add(v)
                stack.addAll(graph[v] ?: emptyList())
            }
        }
        return visited
    }
    
    val connected = findConnected(0)
    println(connected)  // {0, 1, 2, 3}
}
Kotlin - Weighted graph with transformations
fun main() {
    // Weighted graph: Map>>
    val weightedGraph = mapOf(
        "A" to listOf("B" to 4, "C" to 2),
        "B" to listOf("A" to 4, "C" to 1, "D" to 5),
        "C" to listOf("A" to 2, "B" to 1, "D" to 8),
        "D" to listOf("B" to 5, "C" to 8)
    )
    
    // Get all edges with weights
    val allEdges = weightedGraph.flatMap { (node, edges) ->
        edges.map { (neighbor, weight) -> Triple(node, neighbor, weight) }
    }
    println(allEdges)
    // [(A, B, 4), (A, C, 2), (B, A, 4), (B, C, 1), (B, D, 5), ...]
    
    // Find minimum edge weight
    val minWeight = allEdges.minOfOrNull { it.third }
    println(minWeight)  // 1
    
    // Sum of all weights per node
    val weightsPerNode = weightedGraph.mapValues { (_, edges) ->
        edges.sumOf { it.second }
    }
    println(weightsPerNode)  // {A=6, B=10, C=11, D=13}
    
    // Filter nodes by total weight
    val heavyNodes = weightsPerNode.filterValues { it > 10 }.keys
    println(heavyNodes)  // [C, D]
}
