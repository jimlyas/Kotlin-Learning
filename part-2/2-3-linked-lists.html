2.3 Linked Lists

Learn about singly and doubly linked lists with O(1) insertions/deletions and real-world applications.

What is a Linked List?

A linked list is a linear data structure where elements (nodes) are connected via pointers. Each node contains data and a reference to the next node.

Key Properties:

• O(1) insertion/deletion at head or tail

• O(n) random access by index

• No wasted memory from pre-allocation

Real-World Use Cases

Music Playlists

Spotify, Apple Music use linked lists for next/previous track navigation.

Undo/Redo Operations

Text editors use doubly linked lists to traverse edit history.

Browser History

Back/forward navigation in browsers implemented with doubly linked lists.

Blockchain

Cryptocurrency blockchains use linked lists for transaction chains.

Linked List Implementation

Kotlin
class ListNode(var value: Int) {
    var next: ListNode? = null
}

fun reverseList(head: ListNode?): ListNode? {
    var prev: ListNode? = null
    var current = head
    
    while (current != null) {
        val next = current.next
        current.next = prev
        prev = current
        current = next
    }
    return prev
}

Functional Operations with MutableList

In Kotlin, LinkedList implements the List interface, so it supports all functional operations available on collections.

Kotlin - LinkedList operations
fun main() {
    val linkedList = linkedListOf(1, 2, 3, 4, 5)
    
    // Standard List operations (O(n) for linked list)
    val first = linkedList.first  // 1
    val last = linkedList.last   // 5
    val containsThree = 3 in linkedList  // true
    
    // addFirst, addLast (O(1))
    linkedList.addFirst(0)
    linkedList.addLast(6)
    
    // removeFirst, removeLast (O(1))
    linkedList.removeFirst()
    linkedList.removeLast()
    
    // map, filter work but traverse the list
    val doubled = linkedList.map { it * 2 }
    val evens = linkedList.filter { it % 2 == 0 }
    
    // Chaining operations
    val result = linkedList
        .filter { it > 2 }
        .map { it * 10 }
        .reduce { acc, n -> acc + n }
    
    println(result)  // (3+4+5) * 10 = 120
}

LeetCode: Reverse Linked List

Easy

Given the head of a singly linked list, reverse the list and return the reversed list.

View on LeetCode

Use three pointers: prev, current, and next. Iterate through the list, reversing each pointer. Think of it like "rewiring" the connections.

Kotlin Solution
fun reverseList(head: ListNode?): ListNode? {
    var prev = null
    var curr = head
    while (curr != null) {
        val nextTemp = curr.next
        curr.next = prev
        prev = curr
        curr = nextTemp
    }
    return prev
}
