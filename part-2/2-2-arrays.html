2.2 Arrays

Learn about arrays as contiguous memory structures with O(1) random access and real-world applications.

What is an Array?

An array is a contiguous block of memory storing elements of the same type. Each element can be accessed in O(1) time using its index.

Key Properties:

• Fixed size in most languages (Kotlin uses List for dynamic arrays)

• O(1) random access by index

• O(n) insertion/deletion at arbitrary positions

Real-World Use Cases

Image Processing

Pixel data stored in 2D arrays. Used in Android Canvas, image filters, and graphics rendering.

GPU Buffers

Vertex arrays in OpenGL/Vulkan for rendering 3D graphics.

Lookup Tables

Precomputed values for trigonometry, conversion tables. Fast O(1) lookups.

Game Boards

Chess boards, Tic-tac-toe, Sudoku grids stored as 2D arrays.

Arrays in Kotlin

Kotlin
fun main() {
    val numbers = intArrayOf(1, 2, 3, 4, 5)
    
    println(numbers[0])      // 1 (O(1) access)

}

Functional Operations

Kotlin arrays and lists support powerful functional operations for data transformation and analysis.

Kotlin - map, filter, reduce
fun main() {
    val numbers = listOf(1, 2, 3, 4, 5)
    
    // map - transform each element
    val doubled = numbers.map { it * 2 }
    println(doubled)  // [2, 4, 6, 8, 10]
    
    // filter - keep elements matching condition
    val evens = numbers.filter { it % 2 == 0 }
    println(evens)  // [2, 4]
    
    // reduce - accumulate all elements
    val sum = numbers.reduce { acc, n -> acc + n }
    println(sum)  // 15
    
    // fold - reduce with initial value
    val product = numbers.fold(1) { acc, n -> acc * n }
    println(product)  // 120
    
    // any, all, none - check conditions
    println(numbers.any { it > 3 })  // true
    println(numbers.all { it > 0 })  // true
    println(numbers.none { it < 0 })  // true
    
    // find, first, last
    println(numbers.find { it > 2 })  // 3
    println(numbers.first { it > 3 })  // 4
    
    // sorted, reversed, distinct
    println(listOf(3, 1, 2).sorted())  // [1, 2, 3]
    println(listOf(1, 2, 2, 3).distinct())  // [1, 2, 3]
}
Kotlin - flatMap, groupBy, associate
fun main() {
    val words = listOf("hello", "world")
    
    // flatMap - map then flatten
    val chars = words.flatMap { it.toList() }
    println(chars)  // [h, e, l, l, o, w, o, r, l, d]
    
    // associate - create map from list
    val pairs = listOf("a", "b", "c")
    val map = pairs.associateWith { it.uppercase() }
    println(map)  // {a=A, b=B, c=C}
    
    // groupBy - group elements by key
    val numbers = listOf(1, 2, 3, 4, 5)
    val grouped = numbers.groupBy { if (it % 2 == 0) "even" else "odd" }
    println(grouped)  // {odd=[1, 3, 5], even=[2, 4]}
    
    // chunked, windowed
    val data = listOf(1, 2, 3, 4, 5)
    println(data.chunked(2))  // [[1, 2], [3, 4], [5]]
    println(data.windowed(3))  // [[1, 2, 3], [2, 3, 4], [3, 4, 5]]
}

LeetCode: Two Sum

Easy

Given an array of integers nums and an integer target, return the indices of the two numbers that add up to target. You may assume each input has exactly one solution.

View on LeetCode

Think about what complement you need for each number. For each element, check if its complement (target - current) exists in the array using a HashMap for O(1) lookup.

Kotlin Solution
fun twoSum(nums: IntArray, target: Int): IntArray {
    val map = mutableMapOf<Int, Int>()
    for ((index, num) in nums.withIndex()) {
        val complement = target - num
        if (map.containsKey(complement)) {
            return intArrayOf(map[complement]!!, index)
        }
        map[num] = index
    }
    return intArrayOf()
}
