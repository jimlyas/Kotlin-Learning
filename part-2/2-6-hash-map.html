2.6 Hash Map

Learn about key-value storage with O(1) average lookup and practical applications.

What is a Hash Map?

A Hash Map stores key-value pairs and uses a hash function to compute an index into an array of buckets, enabling O(1) average-case lookup, insertion, and deletion.

Time Complexity:

• Average case: O(1) for get/put operations

• Worst case: O(n) due to hash collisions (rare with good hash function)

Real-World Use Cases

Android SharedPreferences

Stores app settings and preferences as key-value pairs.

Caching Systems

Redis, in-memory caches use hash maps for O(1) data retrieval.

Database Indexing

Hash indexes provide fast lookups in NoSQL databases.

Deduplication

Finding unique elements in large datasets.

LeetCode: Valid Anagram

Easy

Given two strings s and t, return true if t is an anagram of s, and false otherwise. An Anagram is a word formed by rearranging the letters of a different word.

View on LeetCode

Count the frequency of each character in both strings. If the frequency maps are equal, the strings are anagrams.

Kotlin Solution
fun isAnagram(s: String, t: String): Boolean {
    if (s.length != t.length) return false
    
    val count = HashMap<Char, Int>()
    for (c in s) {
        count[c] = count.getOrDefault(c, 0) + 1
    }
    for (c in t) {
        count[c] = count.getOrDefault(c, 0) - 1
        if (count[c]!! < 0) return false
    }
    return true
}

Hash Map Functional Operations

Kotlin's Map and MutableMap provide powerful operations for transforming and analyzing key-value data.

Kotlin - Map operations
fun main() {
    val scores = mapOf(
        "Alice" to 95,
        "Bob" to 87,
        "Charlie" to 92
    )
    
    // Access with defaults
    println(scores["Alice"])           // 95
    println(scores.getOrDefault("Unknown", 0))  // 0
    println(scores.getOrElse("X") { -1 })      // -1
    
    // Keys and values iteration
    for (name in scores.keys) {
        println(name)
    }
    for (score in scores.values) {
        println(score)
    }
    
    // entries - access both key and value
    for ((name, score) in scores) {
        println("$name: $score")
    }
}
Kotlin - Map transformations
fun main() {
    val data = mapOf("a" to 1, "b" to 2, "c" to 3)
    
    // mapKeys, mapValues
    val upperKeys = data.mapKeys { it.key.uppercase() }
    println(upperKeys)  // {A=1, B=2, C=3}
    
    val doubled = data.mapValues { it.value * 2 }
    println(doubled)  // {a=2, b=4, c=6}
    
    // filter keys, values, entries
    val filteredKeys = data.filterKeys { it > "a" }
    println(filteredKeys)  // {b=2, c=3}
    
    val filteredValues = data.filterValues { it > 1 }
    println(filteredValues)  // {b=2, c=3}
    
    val filteredEntries = data.filter { it.value > 1 }
    println(filteredEntries)  // {b=2, c=3}
    
    // MutableMap operations
    val mutable = mutableMapOf("x" to 1)
    mutable["y"] = 2
    mutable.put("z", 3)
    mutable.remove("x")
    mutable.getOrPut("new") { 10 }  // adds if not exists
    println(mutable)  // {y=2, z=3, new=10}
}
Kotlin - Map to collection conversions
fun main() {
    val map = mapOf("a" to 1, "b" to 2)
    
    // toList, toSet
    val pairs = map.toList()  // List>
    val keys = map.keys.toList()
    val values = map.values.toList()
    
    // Flatten nested maps
    val nested = mapOf(
        "group1" to mapOf("a" to 1),
        "group2" to mapOf("b" to 2)
    )
    val flattened = nested.flatMap { it.value.entries }
        .associate { it.key to it.value }
    
    // toMap for merging
    val map1 = mapOf("a" to 1)
    val map2 = mapOf("b" to 2)
    val merged = map1 + map2  // {a=1, b=2}
}
