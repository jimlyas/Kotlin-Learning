3.3 Recursion

Learn about recursive problem-solving with base cases and recursive calls, including Fibonacci sequence examples.

What is Recursion?

Recursion is a technique where a function calls itself to solve smaller subproblems. Every recursive solution needs:

• Base Case: The condition that stops the recursion

• Recursive Case: The function calls itself with a smaller input

Fibonacci Implementation

Kotlin
// Recursive Fibonacci - O(2^n) time
fun fibRecursive(n: Int): Int {
    if (n <= 1) return n
    return fibRecursive(n - 1) + fibRecursive(n - 2)
}

// Iterative Fibonacci - O(n) time
fun fibIterative(n: Int): Int {
    if (n <= 1) return n
    var a = 0, b = 1
    for (i in 2..n) {
        val temp = a + b
        a = b
        b = temp
    }
    return b
}

LeetCode: Fibonacci Number

Easy

Given n, calculate F(n), where F(n) is the Fibonacci number calculated as: F(0) = 0, F(1) = 1, F(n) = F(n - 1) + F(n - 2) for n > 1.

View on LeetCode

Start with base cases F(0) = 0 and F(1) = 1. For n > 1, return F(n-1) + F(n-2). Consider using iteration for better performance.

Kotlin Solution
fun fib(n: Int): Int {
    if (n <= 1) return n
    var a = 0, b = 1
    for (i in 2..n) {
        val temp = a + b
        a = b
        b = temp
    }
    return b
}
