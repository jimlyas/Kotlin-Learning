1.5 Control Flow

Master Kotlin's control structures: if expressions, when expressions, for loops, and while loops.

If Expressions

In Kotlin, if is an expression that returns a value. Unlike Java, there's no ternary operator because if serves that purpose.

Kotlin
fun main() {
    val a = 10
    val b = 20
    
    val max = if (a > b) a else b
    println(max)  // 20
    
    val result = if (a > b) {
        println("a is greater")
        a
    } else {
        println("b is greater")
        b
    }
}

When Expressions

when is Kotlin's replacement for switch statements but much more powerful. It can match on conditions, types, and ranges.

Kotlin
fun main() {
    val x = 2
    
    when (x) {
        1 -> println("One")
        2 -> println("Two")
        3, 4 -> println("Three or Four")
        in 5..10 -> println("Between 5 and 10")
        else -> println("Unknown")
    }
}

when can also be used as an expression:

Kotlin
fun describe(obj: Any): String = when (obj) {
    1 -> "One"
    "Hello" -> "Greeting"
    is Long -> "Long integer"
    !is String -> "Not a string"
    else -> "Unknown"
}

For Loops

Use for to iterate over ranges, collections, or arrays.

Kotlin
fun main() {
    val items = listOf("apple", "banana", "cherry")
    
    for (item in items) {
        println(item)
    }
    
    for (i in 1..5) {
        print(i)  // 12345
    }
    
    for (i in 5 downTo 1 step 2) {
        print(i)  // 531
    }
}

While Loops

while and do-while loops work similarly to other languages.

Kotlin
fun main() {
    var x = 5
    while (x > 0) {
        print(x)
        x--
    }  // 54321
    
    var y = 1
    do {
        print(y)
        y++
    } while (y <= 3)  // 123
}
