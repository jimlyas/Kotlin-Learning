1.2 First Program

Write your first Kotlin program and learn about the main function, print statements, and reading user input.

Your First Kotlin Program

Every Kotlin program needs a main function as the entry point. Let's start with the classic "Hello, World!" program.

Kotlin main.kt 
fun main() {
    println("Hello, World!")
}

Run this program and you'll see Hello, World! printed to the console. Simple, right?

Printing Output

print() prints without a newline, while println() adds a newline at the end.

Kotlin 
fun main() {
    print("Hello ")
    print("World")
    println()  // Empty line
    println("New line starts here")
}

You can also print numbers and other types directly:

Kotlin 
fun main() {
    println(42)
    println(3.14159)
    println(true)
}

Reading Input

Use readln() to read user input from the console:

Kotlin 
fun main() {
    println("Enter your name:")
    val name = readln()
    println("Hello, $name!")
}
