1.7 Object-Oriented Programming

Learn about classes, objects, inheritance, interfaces, and data classes in Kotlin.

Classes and Objects

Classes in Kotlin are declared with the class keyword. The primary constructor is part of the class header.

Kotlin
class Person(val name: String, var age: Int) {
    fun greet() = println("Hi, I'm $name")
}

fun main() {
    val person = Person("Alice", 30)
    person.greet()
    person.age = 31
}

Inheritance

Classes are final by default. Use open to allow inheritance.

Kotlin
open class Animal(val name: String) {
    open fun sound() = println("...")
}

class Dog(name: String) : Animal(name) {
    override fun sound() = println("Woof!")
}

fun main() {
    val dog = Dog("Buddy")
    dog.sound()  // Woof!
}

Data Classes

Data classes automatically generate equals(), hashCode(), toString(), and copy().

Kotlin
data class User(val name: String, val email: String)

fun main() {
    val user1 = User("Alice", "alice@example.com")
    val user2 = user1.copy(name = "Bob")
    
    println(user1)  // User(name=Alice, email=alice@example.com)
    println(user2)  // User(name=Bob, email=alice@example.com)
    println(user1 == user2)  // false
}
