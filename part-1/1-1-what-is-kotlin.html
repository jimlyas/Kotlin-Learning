1.1 What is Kotlin?

Discover Kotlin, a modern programming language that runs on the JVM and is officially supported by Google for Android development.

A Brief History

Kotlin was created by JetBrains, the company behind IntelliJ IDEA, and first released in 2011. The name comes from Kotlin Island near St. Petersburg, Russia, inspired by Java (named after Java coffee).

In 2017, Google announced first-class support for Kotlin on Android. In 2019, Google made Kotlin the preferred language for Android development. Today, Kotlin is used by millions of developers for Android, backend, web, and even iOS development.

Key Features

Concise: Kotlin reduces boilerplate code significantly compared to Java. No need for semicolons, getters/setters, or verbose type declarations.

Safe: Kotlin eliminates null pointer exceptions through its nullable type system. The compiler prevents NullPointerException at compile time.

Interoperable: Kotlin is 100% interoperable with Java. You can call Java code from Kotlin and vice versa seamlessly.

Tool-Friendly: Kotlin works with all major IDEs including IntelliJ, Android Studio, and VS Code.

Coroutines: Built-in support for asynchronous programming with lightweight coroutines instead of threads.

Kotlin vs Java

Here's a comparison of the same functionality in Java vs Kotlin:

Java 
public class HelloWorld {
    public static void main(String[] args) {
        String name = "World";
        System.out.println("Hello, " + name + "!");
        
        if (name != null) {
            System.out.println(name.length());
        }
    }
}
Kotlin 
fun main() {
    val name: String = "World"
    println("Hello, $name!")
    
    println(name?.length)
}

Where is Kotlin Used?

Android Development

Official language for Android. Used by Pinterest, Trello, Netflix, and many more.

Backend / Server

Build server-side apps with Spring Boot, Ktor, or Javalin. Used by Square, Atlassian.

Web Development

Kotlin/JS for frontend, Kotlin/Wasm for WebAssembly.

Multiplatform

Kotlin Multiplatform (KMP) shares code between iOS, Android, web, and desktop.

Data Science

Works with Jupyter notebooks, Apache Spark, and KotlinDL for machine learning.

Scripting

Use Kotlin as a scripting language with .kts files for Gradle and more.
